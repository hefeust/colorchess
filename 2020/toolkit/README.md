
# Colorchess Toolkit

## Description

Colorchess toolkit is part of the Colorchess project. This is a set of useful data manipulation structures.

## Functions reference

* create_mwc: creates a Multply-With-Carry Pseudo Random Numbers Generator
* intersect: computes the intersection of two arrays
* create_bidiamp: creates a bidirectional map object
* create_bmp: creates a block meory pool for storing in vivo data
* create_trie: creates a trie tree based data structure
